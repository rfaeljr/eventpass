Fluxo de Pareamento de Dispositivo (Device Pairing)
Admin gera QR Code no painel web → App Android lê → dispositivo fica vinculado à conta e recebe token de acesso único.
1 Admin clica em "+ Dispositivo" no painel
2 API gera JWT de pareamento (uso único, expira em 10min)
3 QR Code exibido na tela do computador
4 App Android lê o QR Code
5 App troca o JWT por Device Token permanente
6 Dispositivo aparece como 🟢 Online no painel
7 Admin associa o device a um evento ativo

Multi-tenant: cada cliente Admin tem seus dados isolados no banco (schema por tenant ou tenant_id). O Device Token é diferente do JWT do usuário — ele identifica o hardware da portaria de forma permanente e independente de sessão.
